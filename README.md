## Description

의미있는 선물? 1분이면 충분합니다.
오프라인에 존재했던 롤링페이퍼 서비스를 온라인으로 옮겨왔습니다.
동적이고, 예쁜 롤링페이퍼를 친구들과 손쉽게 만들어, 친구에게 친구만을 위한 사이트를 선물하세요.

## Dev stack

- React.js with JavaScript
- SSL Framework로 Next.js 사용
- UI Framework로 material-ui 사용

## Deploy

- vercel
